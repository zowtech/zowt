2. **Nome:** `zowt-website`
3. **Público:** ✅ (obrigatório para gratuito)
4. **Create repository**

### 2. Upload Código
1. **Download:** Replit como ZIP
2. **Extrair:** Pasta `scripts/github-pages-deploy/`
3. **Upload:** Arquivos no GitHub (drag & drop)
4. **Commit:** "Initial commit"

2. **Source:** "GitHub Actions"
